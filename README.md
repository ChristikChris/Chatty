# Chatty

A little tool that lives within the chat window and reminds the associate to answer the seller just in time.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. **[Click here to install Chatty](https://github.com/ChristikChris/Chatty/raw/main/Chatty.js)** - Tampermonkey will prompt you to install, or copy-paste the URL into Tampermonkey:
